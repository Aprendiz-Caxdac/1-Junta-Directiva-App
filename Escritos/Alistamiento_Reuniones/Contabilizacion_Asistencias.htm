Contabilización Asistencias

Este es un paso crucial y automatizado. Después de tener la lista de "Asistentes por Reunión", este módulo se usa para generar la transacción contable. Al ejecutarlo, el sistema crea un código o número de contabilización que se envía al área de Contabilidad para que ellos procedan con el pago. Es el paso que convierte la asistencia en una obligación de pago.

Se marca el tipo de reunión, fecha y se oprime la tecla contabilizar.

Se confirma y seguidamente aparece la ventana donde se le indica el número de contabilización.
