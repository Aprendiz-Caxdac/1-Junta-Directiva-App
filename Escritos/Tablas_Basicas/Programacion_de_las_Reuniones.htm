Programación de las Reuniones

ste sub-módulo se usa para crear un registro oficial de que una reunión tuvo lugar. Es el punto de partida para generar un pago. Aquí se registra:

  • El tipo de comité (ej. Comité de Inversiones).
  • La fecha de la reunión.
  • La hora de inicio.
