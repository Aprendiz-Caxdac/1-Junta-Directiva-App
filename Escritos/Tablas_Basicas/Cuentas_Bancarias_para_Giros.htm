Cuentas Bancarias para Giros

Este módulo complementa la información de "Asistentes a las Reuniones". Mientras que el anterior crea el perfil básico, aquí se asocia específicamente a cada persona con el comité o junta a la que pertenece y se confirman sus datos de pago. El proceso es:

  1. Seleccionar a la persona (usando la cédula).
  2. Marcar el comité o junta directiva al que pertenece.
  3. Verificar o registrar nuevamente la información bancaria (banco, tipo de cuenta, número de cuenta) para ese comité específico.
