Actualización Base de Datos

Se ingresa a tablas básicas. Y se escoge Asistentes a las Reuniones.

Se actualiza cada columna. Si es una persona nueva, se le debe solicitar certificación bancaria, copia del documento de identidad y el RUT. Solicitar mediante correo a la coordinadora jurídica, verificar en la lista OFAC informando nombre completo y número de cédula. Enviar el RUT y la certificación OFAC al director de contabilidad, para que lo ingrese en la base de datos terceros.

Efectuada la novedad en contabilidad, se procede a actualizar el cuadro.

Se inserta clase documento, número de cédula, el nombre y cada uno de los datos correspondientes y se graba.

Una vez actualizada la parte de los asistentes, se ingresa a Cuentas Bancarias para Giros.

De la misma manera, se actualiza cada columna con los datos suministrados y se graba.

Esta actualización se debe hacer cada vez que ingresa una persona nueva, si notifican cambio de cuenta bancaria, si la persona cambia de comité o si se retira.
